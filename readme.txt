SocialShare is a simple plugin that enables you to add social share buttons to all of your posts and/or pages easily.

== Description ==

You are able to choose different layouts for buttons from plugin panel. 
Plugin also offers different locations to show sharing buttons.
A self-explanatory administration screen will help you to configure plugin in no time!

== Installation ==

= Automatic =
* Click the ‘Plugins’ tab in WordPress admin view
* Click the ‘Add New’ button
* Search for SocialShare’
* Click ‘Install Now’
* Activate the plugin
* Navigate to ‘Installed Plugins’
* Click ‘Settings’
* Configure Options
* Click the Save Button

= FTP =
* Upload SocialShare to the `/wp-content/plugins/` directory
* Activate the plugin
* Navigate to ‘Installed Plugins’
* Click ‘Settings’
* Configure Options
* Click the Save Button

= Upload =
* Upload the downloaded zip file on the ‘Add New’ plugins screen (see the ‘Upload’ tab) in your WordPress admin view
* Activate the plugin
* Navigate to ‘Installed Plugins’
* Click ‘Settings’
* Configure Options
* Click the Save Button
